apt 套件探索

下載套件

下載「apt」這個套件。

$ apt download apt

解開套件

解開「apt_1.2.10ubuntu1_amd64.deb」

$ dpkg -x apt_1.2.10ubuntu1_amd64.deb apt

觀看資料夾結構

$ tree apt
